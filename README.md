# MacShnaknels

Адаптивний навчальний вебзастосунок ресторану швидкого харчування, створений на HTML, CSS та JavaScript.

## Можливості
- адаптивне меню з фільтрами
- бургер-навігація для мобільних пристроїв
- інтерактивний кошик на JavaScript
- плавні анімації появи елементів
- логічна файлова структура

## Структура проєкту
- `index.html` — головна сторінка
- `css/style.css` — стилі
- `js/script.js` — логіка сайту
- `assets/images/` — зображення

## Запуск
1. Відкрити папку проєкту.
2. Запустити файл `index.html` у браузері.
3. За потреби замінити зображення у папці `assets/images` на власні.
